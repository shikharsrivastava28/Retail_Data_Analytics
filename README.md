# 📊 Retail Customer Behavior Data Analytics Project

A complete end-to-end **Data Analytics Project** focused on transforming raw retail customer data into meaningful business insights using **Python, SQL, and Power BI**.

This project demonstrates the full analytics workflow followed in real-world organizations — from data loading and cleaning to visualization and reporting.

---

## 🚀 Project Overview

The objective of this project is to analyze customer shopping behavior and generate actionable business insights.

The workflow includes:

- Loading dataset using **Python**
- Performing **data cleaning and preprocessing**
- Conducting **Exploratory Data Analysis (EDA)**
- Running business-focused **SQL queries on MySQL**
- Building an interactive **Power BI dashboard**
- Preparing a detailed **project report**
- Creating a professional **presentation using Gamma**

This project is designed to reflect an industry-level data analyst workflow.

## 📂 Project Workflow

### 1. Data Loading
The dataset is loaded into Python using Pandas for further analysis and transformation.

### 2. Data Cleaning
Performed data preprocessing tasks such as:

- Handling missing values
- Removing duplicates
- Fixing data types
- Formatting columns
- Standardizing values

### 3. Exploratory Data Analysis (EDA)
Analyzed customer behavior and sales trends using Python visualizations.

Key insights include:

- Purchase trends
- Customer demographics
- Product category performance
- Spending patterns
- Revenue analysis

### 4. SQL Analysis
Loaded the cleaned dataset into **MySQL Server** and executed SQL queries to solve business problems such as:

- Top-selling products
- High-value customers
- Sales by category
- Monthly revenue trends
- Customer segmentation

### 5. Dashboard Creation
Built an interactive **Power BI dashboard** to present insights through:

- KPI cards
- Bar charts
- Pie charts
- Trend analysis
- Filters and slicers

### 6. Reporting & Presentation
Created:

- A detailed **project report**
- A professional **PPT presentation using Gamma**

for business stakeholders and portfolio presentation.
